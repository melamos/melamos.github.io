Content design & UI writing

Content design shapes how users experience and understand products through microcopy, UI patterns, and interaction flows. I establish scalable UI content standards, build real-time content design capacity across organizations, and apply content design principles to reduce cognitive load in complex workflows.

Establishing content design standards (McAfee)

I created comprehensive UI text guidelines for McAfee's security products covering voice and tone principles, button label patterns, error message frameworks, and microcopy standards. The style guide established consistent UI language across the product suite and became the reference for Product, Design, and Engineering teams implementing new features.

Building real-time content design capacity (Instabase)

Recognizing the need for real-time content design guidance in an Engineering-driving environment, I founded an #ask-word-nerds Slack channel where implementation teams could request guidance on UI text, error messages, and microcopy on demand. The channel became the authoritative resource for content design questions, preventing inconsistent implementations and building content design awareness across the organization. The channel handled complex decisions like error message clarity, tooltip strategy, and form validation messaging.

Real-time content design collaboration addressing error message clarity and user experience concerns.

Applying content design principles to complex workflows (Instabase)

As the authoritative voice for the enterprise automation product, I designed clear, user-centered UI text for complex AI document processing workflows, including getting started guidance, project setup flows, accuracy testing interfaces, and key management. I applied content design principles and best practices to reduce cognitive load and guide users through technical tasks requiring multiple steps and configuration decisions.
